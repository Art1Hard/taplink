# Taplink

Простой таплинк‑проект с актуальными ссылками для связи, портфолио и моих площадок.

## Зачем этот проект

Этот репозиторий — единая точка входа:

* быстро связаться со мной;
* посмотреть портфолио и проекты;
* перейти в соцсети и другие сервисы;
* использовать ссылку как визитку.

Проект сделан лёгким, без лишней логики и перегруза — открывается быстро и одинаково хорошо работает на десктопе и мобильных устройствах.

## Что внутри

* Страница со всеми актуальными ссылками
* Минималистичный дизайн
* Адаптивная вёрстка
* Простая структура проекта

## Технологии

* HTML / SASS / JavaScript
* Gulp / Webpack / Plagins

## Использование

1. Клонировать репозиторий
2. Установить зависимости (если есть)
3. Запустить локально или задеплоить на любой хостинг (Vercel, Netlify, GitHub Pages и т.д.)

## Кастомизация

* Меняй ссылки в одном месте
* Легко добавить новые блоки
* Можно использовать как шаблон под другие проекты
